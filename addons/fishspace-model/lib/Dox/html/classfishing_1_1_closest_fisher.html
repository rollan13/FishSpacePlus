This method contains the algorithm for catching fish of the ClosestFisher type.

Algorithm: Traverse the area. Check if (1) the cell falls inside the maximum distance possible, (2) the cell has enough biomass to satisfy the threshold but not fall below the baseline, (3) the cell is closest to the home port so far. The threshold of each fisher is the maximum threshold of fish caught per time step. The threshold is divided between two fish biomass based on the biomass1_maxratio (a decimal number from 0 to 1). For each time step, the fisher will start from his home port (home_x and home_y).

Parameters
fish_biomassA matrix of the first type of fish biomass. This matrix size (N x M) should be the same as fish_biomass2.
fish_biomass2A matrix of the second type of fish biomass. This matrix size (N x M) should be the same as fish_biomass1.
